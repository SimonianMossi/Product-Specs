Product specs:
Implement a crawler that creates a database of product specs.

The Crawler

Our Crawler was developed in Pycharm. We took advantage of the scrapy library and the web pages we crawled were two of the monts famous electronics stores in Greece, Plaisio and Kotsovolos.

The Data

We programmed the Crawler to store the data in csv files, which we then combined and modified in excel in order to later Visualize them.

The Visualizations

The Visualization of the Data was done using Tableau. Every chart is interactive, so make sure to play around with the filters on the side. The table in the Raw Data section is our most powerfull tool though. There, you can select the exact specs of a laptop you want and the table will show you if those two stores sell any that fit your needs.

The Webpage

This web page was developed in Visual Studio Code using HTML, CSS and JavaScript.
